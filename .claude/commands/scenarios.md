description: Derive motivating scenarios (narratives) from the input pack — the top of the derivation chain.
---

Use the **scenario-and-cq-agent** (`.claude/agents/scenario-and-cq-agent.md`). From the typed input
pack, write short **motivating scenarios** — real situations the ontology must support, as narratives,
not class lists. Ground every scenario in the input; if the input is thin, say so and recommend
`/interview`.

These scenarios justify the ontology and are the source the CQs are extracted from (`/elicit-cqs`
next). Read `derivation-chain.md`. Do not draw classes.
